## 🔄 API ENDPOINTS À CRÉER

### 1. POST /facebook/post-product
```json
{
  "product_type": "genius",
  "color": "60a",
  "template": "vedette",
  "image_url": "https://..."
}
```

### 2. POST /facebook/post-testimonial
```json
{
  "salon_name": "Salon Carouso",
  "quote": "Mes clientes adorent...",
  "city": "St-Georges"
}
```

### 3. POST /facebook/post-reel
```json
{
  "video_url": "https://...",
  "type": "transformation",
  "title": "De court à long en 2h",
  "hashtags": ["Transformation", "Extensions"]
}
```

### 4. POST /facebook/post-story
```json
{
  "media_type": "photo",
  "media_url": "https://...",
  "caption": "En coulisses..."
}
```

### 5. POST /facebook/post-carousel
```json
{
  "theme": "tendances_couleur",
  "images": ["url1", "url2", "url3"],
  "captions": ["Blond Platine", "Châtain Miel", "Noir Soie"]
}
```

### 6. GET /facebook/calendar
```json
// Retourne le prochain post à publier selon le calendrier
```

### 7. POST /facebook/auto-post
```json
{
  "mode": "smart",
  "follow_calendar": true
}
// Publie automatiquement le prochain contenu selon le calendrier
```

## ⚙️ PERMISSIONS FACEBOOK REQUISES

Pour automatiser tout cela, votre app Facebook doit avoir:

- ✅ `pages_manage_posts` - Publier posts/stories
- ✅ `pages_read_engagement` - Lire métriques
- ✅ `pages_show_list` - Lister vos pages
- ✅ `publish_video` - Publier vidéos/Reels
